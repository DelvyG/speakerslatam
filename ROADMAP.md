# Roadmap - SpeakerLATAM

## Sprint 1 (Semanas 1-2): Fundación
- Setup de monorepo y repositorios
- Setup de Laravel + Filament + Sanctum
- Setup de Next.js + Tailwind + shadcn/ui
- Modelo de datos completo (migraciones)
- Seeders básicos para desarrollo
- Configuración de roles y permisos
- Layout público base de Next (header, footer, fonts)

## Sprint 2 (Semanas 3-4): Directorio público
- Página home (mockup home.png)
- Página directorio con filtros (mockup directorio.png)
- Página de perfil de speaker (mockup perfil-conferencista.png)
- Página "Para Empresas" (mockup seccion-empresas.png)
- API endpoints públicos: /speakers, /speakers/{slug}, /categories
- ISR configurado

## Sprint 3 (Semanas 5-6): Onboarding y membresía
- Página de registro speaker wizard (mockup registro-conferencista.png)
- Integración Stripe para membresía anual
- Sistema de validación admin (Filament)
- Dashboard básico del speaker
- Email transaccional bienvenida

## Sprint 4 (Semanas 7-8): Empresas y concierge
- Formulario de solicitud empresa
- Webhook a Kommo
- Sección Academy / Recursos (mockup academy.png)
- Newsletter signup
- QA general y soft launch

## Fase 2 (Mes 3-4)
- E-commerce de add-ons en dashboard speaker
- Sistema de convocatorias concierge
- Métricas de visualización del perfil

## Fase 3 (Mes 5+)
- Sistema de reseñas
- Calendario de disponibilidad
- App mobile PWA
